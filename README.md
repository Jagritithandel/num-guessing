# num-guessing

A small, fun number-guessing game — "num-guessing". The player must guess a hidden number between 1 and N. After each guess the game tells you whether the number is higher, lower, or correct.

## Features
- Simple command-line interface
- Customizable range (default 1–100)
- Tracks number of attempts
- Great for beginners and kids
Tech
- Python 3.x 

Setup
1. Clone the repo:
 
   git clone https://github.com/Jagritithandel/num-guessing.git
   cd num-guessing

2. Install dependencies

- pip install

3.Run

- python numguess.py
 
##Gameplay example

Guess a number between 1 and 100: 50

Too low! Try again.

Guess a number between 1 and 100: 75

Too high! Try again.

Guess a number between 1 and 100: 63

Correct! You guessed it in 3 tries.
